Munchies: Munchies is a snack recommendation system. We use classification to correlate snack review data, which was provided by Amazon, to snack Adjectives. Users enter their snack adjective selection in the auto-complete search bar, checkboxes, and/or the drop down menu's. Then, the User's input is sent to our server, the server feeds the input to our model built from classifiaction, the model returns snack recommendations results to the android client, and the android client lastly displays the results to the user in list. The user can then click on any snack on the list and a dialog with more information of the snack appears.
